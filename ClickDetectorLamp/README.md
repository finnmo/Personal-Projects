
# Click Detector Lamp
***Author: Finn Morris
Created: 05/11/21
Modified: 05/11/21***

# Description:
A lamp that toggles with a click of your fingers

### Hardware: 
- Arduino Sound Sensor
- Wemos D1 mini
- Lamp
 
# Purpose:
The carefully hidden sound sensor detects a click and sends a request to Homeassistant to toggle the lamp.
